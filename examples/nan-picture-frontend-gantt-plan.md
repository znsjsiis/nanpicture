# nan-picture-frontend 项目开发甘特图

## 📋 项目概述

**项目名称**: nan-picture-frontend（小张云图库前端）  
**开发周期**: 2024年1月1日 - 2024年1月26日（共23天）  
**技术栈**: Vue 3 + TypeScript + Vite, Ant Design Vue, Vue Router, Pinia

## 🎯 核心功能模块

### 1. 用户管理模块

- 用户登录/注册
- 个人信息管理
- 退出登录

### 2. 图片管理模块

- 创建图片（上传图片、填写信息）
- 编辑图片
- 图片列表展示
- 图片详情查看
- 搜索和筛选（按分类、标签）

### 3. 后台管理模块

- 用户管理（管理员专用）
- 图片管理（管理员专用）

### 4. 公共组件

- 全局导航栏（GlobalHeader）
- 图片上传组件（PictureUpload）
- 基础布局（BasicLayout）

## 📊 页面路由结构

```
/ (首页 - 图片列表)
/add_picture (创建/编辑图片)
/user/login (用户登录)
/user/register (用户注册)
/user/profile (个人中心)
/admin/userManage (用户管理 - 管理员)
/admin/pictureManage (图片管理 - 管理员)
/picture/:id (图片详情)
```
