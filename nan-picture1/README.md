# 小张云图库 - 微信小程序

## 项目简介

这是一个基于微信小程序平台开发的云图库应用，用户可以上传、管理和分享图片。

## 技术栈

- 微信小程序原生框架
- Vant Weapp 组件库
- JavaScript ES6+

## 项目结构

```
nan-picture1/
├── components/          # 自定义组件
│   └── bottom-tab/     # 底部导航组件
├── pages/              # 页面文件
│   ├── gallery/        # 图库页面（瀑布流展示）
│   ├── upload/         # 图片上传页面
│   ├── user/           # 用户中心页面
│   ├── login/          # 登录页面
│   └── register/       # 注册页面
├── utils/              # 工具类
│   ├── request.js      # 网络请求封装
│   ├── auth.js         # 用户认证工具
│   └── validator.js    # 表单验证工具
├── assets/             # 静态资源
│   ├── images/         # 图片资源
│   └── icons/          # 图标资源
├── app.js              # 小程序入口文件
├── app.json            # 全局配置
├── app.wxss            # 全局样式
└── package.json        # 项目依赖
```

## 主要功能

### 1. 用户系统

- 用户注册/登录
- 个人信息管理
- 登录状态持久化

### 2. 图片管理

- 图片上传（支持多图）
- 图片浏览（瀑布流布局）
- 图片搜索和筛选
- 图片详情查看

### 3. 界面特色

- 简约现代的设计风格
- 响应式布局适配
- 流畅的交互动效
- 统一的视觉语言

## 开发环境配置

### 1. 安装依赖

```bash
npm install
```

### 2. 微信开发者工具配置

- 打开微信开发者工具
- 导入项目目录
- AppID: wxb8474ffd97f26909
- 项目名称: 小张云图库

### 3. 后端服务配置

确保后端服务运行在 `http://localhost:8123`，API前缀为 `/api`

## 部署说明

### 1. 本地调试

- 在微信开发者工具中直接运行
- 确保后端服务正常运行

### 2. 线上部署

- 构建生产版本
- 上传至微信小程序平台
- 配置服务器域名白名单

## 注意事项

1. 图片上传需要后端支持文件存储
2. 用户认证基于Session机制
3. 瀑布流布局使用双列自适应算法
4. 所有网络请求都有统一的错误处理

## 项目特点

- **现代化UI**: 采用Vant Weapp组件库，界面美观统一
- **响应式设计**: 适配不同屏幕尺寸
- **用户体验**: 流畅的动画效果和交互反馈
- **代码规范**: 结构清晰，易于维护和扩展
- **性能优化**: 图片懒加载，合理的数据分页
