## 常见问题

### Q: 如何添加新的知识库文件？
A: 在 `src/main/resources/knowledge/` 目录下添加 `.md` 或 `.txt` 文件即可。

### Q: 知识库文件不生效怎么办？
A:
1. 检查配置文件中 `enabled: true`
2. 确认文件在正确的目录下
3. 查看启动日志中的初始化信息
4. 执行 `mvn clean compile` 重新编译

### Q: 如何获取豆包 API Key？
A: 访问 https://console.volcengine.com/ark，登录后在密钥管理中创建。

### Q: 云图库支持哪些图片格式？
A: 支持常见的 JPG、PNG、GIF、WebP 等格式，可通过腾讯云 COS 进行存储和管理。

### Q: 如何查看上传图片的详细信息？
A: 通过 API 文档的接口可以查看图片列表、详情、分类等信息。
